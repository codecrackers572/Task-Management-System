# Task Management API

Ye ek Node.js + Express ka backend hai jisme hum tasks create, read, update, delete kar sakte hain.

## Features
- Add new task
- Get all tasks  
- Update task status
- Delete task

## Tech Stack
- Node.js
- Express.js
- MongoDB

## How to Run
1. `npm install`
2. `npm start`
3. Server will run on http://localhost:3000
